# 이미지 프롬프트 — LME 다단계 BOM 원가

## [썸네일]
- 이미지 내 텍스트:
  - 상단 헤드라인: "다단계 BOM 원가, 안전하십니까?"
  - 좌(As-Is): "시세는 매일, 원가표는 어제 그대로"
  - 우(To-Be): "BOM 전체를 꿰뚫는 원가 관리"
- 프롬프트:
  세포아소프트 B2B 지출관리 블로그 썸네일. 회로기판(circuit board) 배경 위,
  다크 네이비(#0F172A) 바탕에 #0747A6 강조만 사용하는 2색 스킴(3색 이상 금지).
  좌우 2분할 대비 구도 — 좌측은 흐트러진 스프레드시트·상승 화살표로 As-Is(문제),
  우측은 노드가 연결된 정돈된 BOM 트리로 To-Be(해결). 중앙에 얇은 구분선.
  Pretendard 굵은 국문 텍스트: 상단 "다단계 BOM 원가, 안전하십니까?",
  좌측 "시세는 매일, 원가표는 어제 그대로", 우측 "BOM 전체를 꿰뚫는 원가 관리".
  정돈된 벡터 다이어그램 톤, 전문적·신뢰감. 다크 배경에 밝은 텍스트로 충분한 대비.
  실사·경쟁사 UI·저작권 소지 이미지 배제. 16:9.

## [본문 이미지 1] — As-Is 심화
- 이미지 내 텍스트:
  - 제목: "원가 이탈을 '사후'에 발견하는 구조"
  - 라벨 3: "전파 경로 안 보임" / "목표원가 사후 확인" / "단가·시세 흩어짐"
- 프롬프트:
  세포아 블로그 본문 다이어그램. 회로기판 배경, 다크 네이비(#0F172A) + #0747A6
  2색 스킴. 상단 제목 "원가 이탈을 '사후'에 발견하는 구조"(Pretendard 굵게).
  가로로 배치된 3개 문제 카드, 각 카드에 Lucide 라인 아이콘(16px 스타일)과
  국문 라벨: "전파 경로 안 보임"(분기하는 화살표 아이콘),
  "목표원가 사후 확인"(늦은 알림/시계 아이콘), "단가·시세 흩어짐"(흩어진 파일 아이콘).
  강조색은 위험 신호에만 절제되게 사용. 정돈된 벡터 톤, 밝은 텍스트로 대비 확보.
  과장 문구·경쟁사 UI 배제. 16:9.

## [본문 이미지 2] — To-Be (SingleSuite 3기능 연결)
- 이미지 내 텍스트:
  - 제목: "BOM 전체를 꿰뚫는 원가 관리"
  - 노드 3: "단가·시세 연동" → "다단계 BOM 원가 전개" → "목표원가 모니터링·알림"
  - 하단: "SingleSuite"
- 프롬프트:
  세포아 블로그 본문 다이어그램. 회로기판 배경, 다크 네이비(#0F172A) + #0747A6
  2색 스킴. 상단 제목 "BOM 전체를 꿰뚫는 원가 관리"(Pretendard 굵게).
  좌에서 우로 연결선(회로 트레이스)으로 이어지는 3개 노드:
  "단가·시세 연동" → "다단계 BOM 원가 전개" → "목표원가 모니터링·알림".
  각 노드는 제품 모듈을 뜻하는 node 형태, Lucide 라인 아이콘. 연결선은 #0747A6 강조.
  하단 중앙에 "SingleSuite" 워드마크. 정돈된 벡터 다이어그램 톤, 전문적·신뢰감,
  다크 배경 밝은 텍스트 대비. 실사·경쟁사 UI·저작권 소지 이미지 배제. 16:9.
