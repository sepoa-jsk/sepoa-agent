# 이미지 프롬프트 — AI가 계정과목까지 추천하는 경비정산

[썸네일]
- 이미지 내 텍스트: "지로영수증만 첨부하면 전표가 완성된다면?" / 보조 텍스트: "AI가 계정과목까지 추천하는 경비정산"
- 프롬프트: 세포아소프트 B2B 지출관리 블로그 썸네일. 회로기판 배경, 다크 네이비(#0F172A)에 #0747A6 강조 2색 스킴. 좌측 As-Is(지로영수증 서류 더미와 수기 전표 아이콘, 흐릿한 라인) / 우측 To-Be(지로영수증 한 장이 회로 라인을 따라 완성된 전표 노드로 연결되는 다이어그램, #0747A6 발광 강조) 대비 구도. Pretendard 굵은 국문 텍스트: "지로영수증만 첨부하면 전표가 완성된다면?" 상단 배치, 하단에 작게 "AI가 계정과목까지 추천하는 경비정산". Lucide 계열 라인 아이콘(영수증, 문서, 체크). 정돈된 벡터 다이어그램 톤, 전문적·신뢰감. 실사·3색 이상·과장 문구 금지.

[본문 이미지 1 — 계정과목 추천 개념 (단락3 배치)]
- 이미지 내 텍스트: "같은 전기요금, 회사마다 다른 계정" / 매핑 라벨: "전북도시가스 → 수도광열비", "KT 인터넷요금 → 통신비", "한국전력 → 전력비", "신문 구독료 → 도서인쇄비·신문대" / 하단 캡션: "과거 전표 데이터를 학습해 우리 회사 기준으로 추천"
- 프롬프트: 다크 네이비(#0F172A) 회로기판 배경 위 벡터 다이어그램. 상단 Pretendard 굵은 국문 타이틀 "같은 전기요금, 회사마다 다른 계정". 좌측에 거래처 노드 4개(전북도시가스, KT 인터넷요금, 한국전력, 신문 구독료), 중앙에 AI 노드(회로 패턴 원형, #0747A6 발광), 우측에 계정과목 노드 4개(수도광열비, 통신비, 전력비, 도서인쇄비·신문대)를 회로 라인으로 연결한 매핑 구조. 하단 캡션 "과거 전표 데이터를 학습해 우리 회사 기준으로 추천". 2색 스킴 유지, Lucide 16px 라인 아이콘, 밝은 텍스트로 대비 확보.

[본문 이미지 2 — 처리 프로세스 (단락4 배치)]
- 이미지 내 텍스트: 단계 라벨 7개: "촬영·업로드 → 증빙 자동 인식 → 증빙유형 분류 → 계정과목 추천 → 적요 생성 → 전표 작성 → 결재 상신" / 하단 캡션: "담당자는 확인·승인만"
- 프롬프트: 다크 네이비(#0F172A) 배경, 은은한 회로기판 패턴. 좌→우로 흐르는 7단계 수평 플로우 다이어그램. 각 단계는 라운드 사각 노드 + Lucide 라인 아이콘(카메라, 스캔, 태그, 폴더, 펜, 문서, 결재 도장), 노드 사이를 회로 라인 화살표로 연결. "계정과목 추천" 노드만 #0747A6로 강조 발광. 노드 라벨은 Pretendard 굵은 국문. 우측 끝에 사람 아이콘과 체크 표시, 하단 캡션 "담당자는 확인·승인만". 2색 스킴, 정돈된 벡터 톤.
