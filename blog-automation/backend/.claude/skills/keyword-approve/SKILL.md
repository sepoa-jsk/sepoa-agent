# 키워드 승인 (Step 2)

## 목적
선택된 제목 기준으로 네이버 검색 노출을 위한 키워드를 정리한다.

## 규칙
- **핵심 키워드 1개** + **보조 키워드 3~5개**로 구성.
- 세포아 제품군 용어(전자구매, SRM, 경비지출, 출장관리, 전자계약, 전자인장,
  SingleSuite)를 우선 후보로 검토.
- 각 키워드가 본문 어디에 들어갈지(제목/첫 문단/소제목) 배치안을 함께 제시.
- 억지 키워드 삽입(키워드 스터핑) 금지.

## 출력 형식
```
[키워드 후보]
- 핵심: ...
- 보조: ..., ..., ...
[배치안]
- 제목: ...
- 첫 문단: ...
- 소제목: ...
```

## [STOP]
키워드 후보와 배치안 제시 후 **반드시 멈춘다.**
사람이 승인(또는 수정 지시)하기 전에는 write-body로 진행하지 않는다.
