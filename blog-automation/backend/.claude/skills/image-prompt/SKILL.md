# 이미지 프롬프트 (Step 4)

## 목적
본문에 맞는 썸네일 1개 + 본문 이미지 프롬프트를 만든다.
반드시 `DESIGN.md`의 시각 규칙을 따른다.

## 규칙 (DESIGN.md 준수)
- **구조**: As-Is(문제) → To-Be(해결) 대비 레이아웃.
- **배경**: 회로기판 모티프로 "AI 기반" 시그널.
- **컬러**: 2색 스킴. Primary `#0747A6`, 다크 `#0F172A` 기준. 3색 이상 혼용 금지.
- **폰트**: Pretendard / Noto Sans KR.
- **이미지 내 텍스트**: 짧고 명료하게. 오탈자·과장 문구 금지.
- 경쟁사 UI 캡처·저작권 소지 이미지 금지.

## 출력 형식
```
[썸네일]
- 이미지 내 텍스트: "..."
- 프롬프트: ...
[본문 이미지 N]
- 이미지 내 텍스트: "..."
- 프롬프트: ...
```
- 경로: `images/YYYY-MM-DD-{슬러그}-prompts.md` 에 저장.

## [STOP]
**이미지 내 텍스트**를 먼저 사람에게 보여주고 승인받는다.
승인 전에는 최종 프롬프트를 확정하지 않으며 evaluator로 넘어가지 않는다.
