# DESIGN.md — 세포아 블로그 이미지 시각 규칙

> 썸네일·본문 이미지의 톤을 매 글마다 일관되게 유지하기 위한 한 장짜리 규칙.
> image-prompt 스킬은 이 파일을 근거로 프롬프트를 만든다.

## 1. Visual Theme & Atmosphere
전문성과 신뢰. B2B 실무자가 "믿을 만하다"고 느끼는 정돈된 톤.
회로기판(circuit board) 모티프로 "AI 기반 지출관리"라는 시그널을 준다.

## 2. Color Palette & Roles
- `--primary: #0747A6`   /* SingleSuite 프라이머리, 강조 */
- `--dark: #0F172A`      /* 배경·사이드바 다크 */
- **2색 스킴 원칙**: 배경 1색 + 강조 1색. 3색 이상 혼용 금지.
- 다크 네이비 CMYK 변환 시 자주색(퍼플) 방지: C100 M68~70 Y8~10 K45~58 기준.

## 3. Typography
- 국문: Pretendard, 대체 Noto Sans KR
- 영문·숫자: Roboto
- 이미지 내 텍스트는 굵고 짧게.

## 4. Layout & Composition
- **As-Is → To-Be 대비 레이아웃**을 기본 구도로.
- 좌(문제) / 우(해결) 또는 상(문제) / 하(해결) 분할.

## 5. Iconography
- Lucide 아이콘 계열, 16px 기준 라인 스타일.
- 제품 모듈은 노드(node) 형태로 표현 가능.

## 6. Imagery & Motif
- 회로기판 배경, AI Agent 개념 노드 연결.
- 실사보다 정돈된 벡터·다이어그램 톤 우선.

## 7. Do's and Don'ts (가드레일)
- ✅ DO: As-Is→To-Be 대비, 2색 스킴, 회로기판 배경, 짧은 텍스트.
- ❌ DON'T: 3색 이상 혼용, 경쟁사 UI 캡처, 과장 문구, 저작권 소지 이미지.

## 8. Accessibility
- 텍스트/배경 대비 충분히. 다크 배경엔 밝은 텍스트.

## 9. 재사용 프롬프트 스니펫
```
세포아소프트 B2B 지출관리 블로그 썸네일.
회로기판 배경, 다크 네이비(#0F172A)에 #0747A6 강조 2색 스킴.
좌측 As-Is(문제) / 우측 To-Be(해결) 대비 구도.
Pretendard 굵은 국문 텍스트: "{승인된 텍스트}".
정돈된 벡터 다이어그램 톤, 전문적·신뢰감.
```
