# DESIGN.md — 세포아 블로그 이미지 시각 규칙

> 썸네일·본문 이미지의 톤을 매 글마다 일관되게 유지하기 위한 한 장짜리 규칙.
> image-prompt 스킬은 이 파일을 근거로 프롬프트를 만든다.

## 1. Visual Theme & Atmosphere
전문성과 신뢰. B2B 실무자가 "믿을 만하다"고 느끼는 정돈된 톤.
밝고 깨끗한 배경 위에 회로기판(circuit board) 모티프를 은은하게 얹어
"AI 기반 지출관리"라는 시그널을 준다.

## 2. Color Palette & Roles
- `--bg: #FFFFFF` 또는 `#F8FAFC`   /* 기본 배경, 흰색·아주 연한 회색 */
- `--primary: #0747A6`             /* SingleSuite 프라이머리, 강조 (밝은 배경 위 파란 포인트) */
- `--ink: #0F172A`                 /* 본문·제목 텍스트, 밝은 배경에서 잘 읽히는 다크 네이비 */
- **2색 스킴 원칙**: 배경 1색(밝은 톤) + 강조 1색(`--primary`). 3색 이상 혼용 금지.
- 텍스트·회로 라인에 다크 네이비를 인쇄용 CMYK로 변환할 때 자주색(퍼플)으로 틀어지지
  않도록 C100 M68~70 Y8~10 K45~58 기준을 지킨다.

## 3. Typography
- 국문: Pretendard, 대체 Noto Sans KR
- 영문·숫자: Roboto
- 이미지 내 텍스트는 굵고 짧게. 밝은 배경 위 다크 네이비(`--ink`)로 표기.

## 4. Layout & Composition
- **프로세스 흐름(파이프라인) 레이아웃**을 기본 구도로 한다.
  왼쪽 **입력**(예: 영수증·서류 등 원본) → 중앙 **AI 처리**(노드·회로 라인) →
  오른쪽 **결과물**(예: 완성된 전표)로 이어지는 하나의 좌→우 흐름 다이어그램.
- 좌우를 "문제/해결"로 대립시키지 않는다. **"As-Is"/"To-Be" 같은 글자 라벨을
  이미지 안에 텍스트로 넣지 않는다.** 도입 후 좋아지는 모습은 흐름의 끝(결과물)이
  깔끔하고 정돈되게 완성되는 장면으로 자연스럽게 드러낸다 — 문구로 문제/해결을
  나누어 설명하지 않는다.
- 흐름의 방향성은 화살표나 이어지는 회로 라인으로 명확히 좌→우로 표현한다.

## 5. Iconography
- Lucide 아이콘 계열, 16px 기준 라인 스타일.
- 제품 모듈은 노드(node) 형태로 표현 가능.

## 6. Imagery & Motif
- 흰색/아주 연한 회색 배경 위에 회로기판 라인을 얹은 구도.
- 회로 라인은 입력→AI 처리→결과물을 잇는 흐름 경로 역할도 겸한다. 연한 파랑
  (`--primary`의 옅은 톤) 또는 옅은 회색으로 은은하게 — 다크 배경의 발광 효과
  대신 가느다란 실선으로 표현.
- 흐름 중앙의 AI 처리 노드만 `--primary`를 선명하게 써 시선을 모으고, 입력·결과물
  노드는 상대적으로 차분한 톤으로 둔다.
- 실사보다 정돈된 벡터·다이어그램 톤 우선.

## 7. Do's and Don'ts (가드레일)
- ✅ DO: 밝은 배경(흰색·연한 회색), 입력→AI 처리→결과물 프로세스 흐름 구도, 2색 스킴, 은은한 회로기판 라인, 짧은 텍스트.
- ❌ DON'T: 다크 배경 기본값으로 사용, "As-Is"/"To-Be" 글자 라벨 사용, 좌우 대립 구도로 쪼개기, 3색 이상 혼용, 경쟁사 UI 캡처, 과장 문구, 저작권 소지 이미지.

## 8. Accessibility
- 텍스트/배경 대비 충분히. 밝은 배경엔 다크 네이비(`--ink`) 텍스트로 대비를 확보한다.

## 9. 재사용 프롬프트 스니펫
```
세포아소프트 B2B 지출관리 블로그 썸네일.
흰색/아주 연한 회색(#F8FAFC) 배경에 #0747A6 강조 2색 스킴.
은은한 회로기판 라인(연한 파랑·회색 실선)을 배경에 얹은 구도.
왼쪽 입력(예: 영수증·서류)에서 시작해 중앙 AI 처리 노드(#0747A6 강조)를 거쳐
오른쪽 완성된 결과물로 이어지는 좌→우 프로세스 흐름 다이어그램.
"As-Is"/"To-Be" 같은 글자 라벨이나 좌우 대립 구도는 넣지 않는다.
Pretendard 굵은 국문 텍스트(다크 네이비 #0F172A): "{승인된 텍스트}".
정돈된 벡터 다이어그램 톤, 밝고 전문적·신뢰감.
```
